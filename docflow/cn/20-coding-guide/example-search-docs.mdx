## 01 场景说明

当你需要查找 Docflow 的 API 文档、参数说明或使用指引时，可以直接向 Agent 提问，它会通过 MCP 服务检索相关内容。

## 02 示例

### 2.1 查询 API 接口

```text
帮我查询 Docflow 文档抽取 API 的接口说明
```

### 2.2 查询参数说明


![查询参数说明](../images/example-search-params.png)
