## 01 推荐工具

请先安装一款支持 MCP 协议的 AI Coding 工具，以下是我们推荐的三款：

<CardGroup cols={3}>
  <Card title="Codex" icon="terminal" href="https://openai.com/index/openai-codex/">
    OpenAI 官方命令行编程助手，支持 MCP 协议和 Skill 系统。
  </Card>
  <Card title="Claude Code" icon="message-bot" href="https://code.claude.com/docs/zh-CN/overview">
    Anthropic 官方 CLI 工具，原生支持 MCP 协议。
  </Card>
  <Card title="Trae" icon="code" href="https://www.trae.cn/">
    字节跳动推出的 AI IDE，支持 MCP 协议。
  </Card>
</CardGroup>

<Note>
  本指引后续步骤将以 **Codex** 为例进行说明。如果你使用其他工具，请参考对应工具的文档进行配置。
