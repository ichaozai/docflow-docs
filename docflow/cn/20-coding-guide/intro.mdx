## 01 什么是 Docflow 文档 Skill

TextIn Docflow 文档 Skill 是一个为 AI Coding Agent 提供的技能插件，让你的编程助手能够：

- **检索文档和内容** — 快速查找 TextIn Docflow 相关的 API 文档和技术资料
- **生成示例代码** — 根据文档自动生成可用的代码片段
- **辅助开发** — 在开发过程中实时获取技术指引和最佳实践
