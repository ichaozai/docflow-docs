---
title: "座標系"
---

座標は 8 つの整数を含む配列で、形式は `[x1, y1, x2, y2, x3, y3, x4, y4]` です。

四角形の 4 つの頂点の座標を表し、時計回りに配置されています。
すなわち：

- 左上：(x1, y1)
- 右上：(x2, y2)
- 右下：(x3, y3)
- 左下：(x4, y4)

座標値は絶対ピクセル値です。座標系は以下のとおりです：

```
画像座標系（原点は左上隅）
┌────────────width───────────> x
│(0,0)
│
│
│      (x1,y1) ────────── (x2,y2)
│         │                     │
height           テキスト領域
│         │                     │
│      (x4,y4) ────────── (x3,y3)
│
v
y
```
