#### フィールドの高度な設定

[ファイルカテゴリフィールドを更新](/api-reference/%E5%8D%98%E4%B8%80%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E3%82%AB%E3%83%86%E3%82%B4%E3%83%AA%E3%83%95%E3%82%A3%E3%83%BC%E3%83%AB%E3%83%89%E3%82%92%E6%9B%B4%E6%96%B0) (single + [batch](/api-reference/%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E3%82%AB%E3%83%86%E3%82%B4%E3%83%AA/%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E3%82%AB%E3%83%86%E3%82%B4%E3%83%AA%E3%81%AE%E3%83%95%E3%82%A3%E3%83%BC%E3%83%AB%E3%83%89%E3%82%92%E6%9B%B4%E6%96%B0)) now support these properties:

| プロパティ | 説明 | 更新時の動作 |
|----------|-------------|-----------------|
| `alias` | Field aliases | omit=no change, `[]`=clear, `["a","b"]`=override |
| `identity` | Export field name | omit=no change, `""`=clear, non-empty=override |
| `multi_value` | Multi-value extraction | omit=no change, `true`/`false`=override |
| `duplicate_value_distinct` | Deduplicate values | Only effective when `multi_value=true` |
| `transform_settings` | Output transform config | Supports datetime / enumerate / regex |
